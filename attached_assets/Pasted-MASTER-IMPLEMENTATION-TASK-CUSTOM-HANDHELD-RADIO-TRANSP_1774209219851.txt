====================================================

Do NOT rewrite app.js.
Do NOT refactor signaling architecture.
Only extend signalingService.js.

FloorControlService responsibilities:

- Track floor holder per channel
- Grant / deny logic
- Automatic release on timeout
- Release on disconnect
- Emergency preemption using existing emergency state
- Pure service — no direct socket emissions

AudioRelayService responsibilities:

- Bind UDP socket (env AUDIO_RELAY_PORT default 5100)
- Authenticate packet sender via session token
- Verify sender holds floor
- Maintain subscriber membership list per talkgroup
- Fan out Opus frames ONLY to subscribers of that talkgroup
- Expose recording tap callback
- Do NOT mix or transcode audio

Packet format:

[session token][channelId uint16][sequence uint16][Opus payload]

Security rules:

- Reject packet if token invalid
- Reject packet if channel unauthorized
- Reject packet if sender does not hold floor
- Remove subscribers on disconnect / leave / timeout

- ptt:request
- ptt:release
- tx:start
